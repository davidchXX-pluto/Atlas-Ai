# Atlas-AI

Atlas-AI is a simple, stateful AI assistant created by David to help non-technical users understand everyday concepts and use technology confidently.

## Features
- Step-by-step explanations
- Remembers user preferences
- Clear, structured responses
- Creator identity stored separately
- Low-cost / free-first architecture

## Tech Stack
- Python
- Groq API (LLMs)
- JSON-based memory

## How to Run
1. Activate virtual environment
2. Run `python main.py`
3. Ask questions in plain English
